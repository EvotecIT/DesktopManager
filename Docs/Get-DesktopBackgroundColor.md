# Get-DesktopBackgroundColor

## SYNOPSIS
Retrieves the current desktop background color.

## DESCRIPTION
Returns the desktop background color as a numeric RGB value.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-DesktopBackgroundColor
```
Gets the current background color.

## OUTPUTS

### System.UInt32
